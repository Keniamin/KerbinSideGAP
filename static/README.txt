=== Installation
1. Install the dependencies: KerbinSide and ContractConfigurator mods.
2. Copy KerbinSideGAP folder to your GameData/ContractPacks folder (create ContractPacks if it is not exist in GameData).
3. (Optional) Install NavUtilities mod and copy KerbinSideRunways.rwy to GameData/KerbalScienceFoundation/NavInstruments/Runways. 

=== Source and updates
Source code and latest releases are available at https://github.com/Keniamin/KerbinSideGAP/releases
